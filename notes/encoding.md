# Encoding


Bit Index | Usage
----------|---------------------------------------------------------------------------------
0         | Color (1 = Black, 0 = White)
1-3       | Character type (1 = Pawn, 2 = Knight, 3 = Bishop, 4 = Rook, 5 = Queen, 6 = King)
4-10      | Position (0-63) (0 = A1, 63 = H8)
